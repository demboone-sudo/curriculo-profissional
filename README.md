# 🚀 Portfólio & Currículo Profissional Online

Seja bem-vindo(a) ao meu currículo online e portfólio técnico, projetado para destacar minha transição de carreira para a área de Tecnologia da Informação. Este projeto foi desenvolvido como trabalho prático para o Centro Universitário Estácio de Vila Velha.

O objetivo principal é apresentar meu ecossistema de aprendizado, minhas competências técnicas em desenvolvimento de software, automação e infraestrutura em nuvem, mapeando de forma transparente minha sólida bagagem profissional prévia.

---

## 🛠️ Tecnologias Utilizadas

Para a estruturação, estilização e dinamismo deste ecossistema, foram aplicadas as seguintes tecnologias fundamentais:

- **HTML5:** Estruturação semântica e organização modular de blocos de conteúdo.
- **CSS3 (Advanced Styles):** Estilização baseada em Flexbox e CSS Grid, efeitos translúcidos de vidro (_Glassmorphism_), animações fluidas de formas orgânicas (_Blob_), efeitos dinâmicos de rotação 3D (_Flip_) e responsividade avançada para dispositivos móveis.
- **JavaScript (ES6):** Manipulação dinâmica do DOM para a execução do efeito interativo de digitação automática (_Typewriter effect_) no cabeçalho.
- **PHP:** Processamento backend estruturado no servidor para manipulação, segurança e validação do formulário de contato.

---

## ⚙️ Funcionalidades Implementadas

1. **Apresentação Profissional Dinâmica:** Efeito de máquina de escrever via JavaScript destacando as principais frentes de atuação como Analista de Sistemas.
2. **Visual Glassmorphism & Caixas de Foco:** Seção de Objetivos Profissionais dividida em três blocos interativos (Front-end, Engenharia de Software e Inovação) estilizados com ícones vetorizados do _Font Awesome_.
3. **Mapeamento de Soft e Hard Skills:** Divisão equilibrada na trajetória profissional destacando a maturidade de 13 anos na área da Saúde (Técnica de Enfermagem), 14 anos no Empreendedorismo (Gestão e Confeitaria MEI) e o momento atual em TI.
4. **Seção de Contato Simétrica:** Layout moderno em duas colunas, alinhando um formulário de contato responsivo com um avatar secundário dinâmico em formato _blob_.
5. **Exportação Facilitada (PDF):** Botão nativo integrado que utiliza a folha de estilo de impressão (`@media print`) para gerar um arquivo PDF limpo e ideal para o mercado de trabalho.
